### DELETING BRANCHS
##### Delete local brach
``` 
git branch -d <brandName>
```

##### Delete remote branch
``` 
git push -d origin <branchName>
```

### CHANGE REMOTE URL
##### Check remote url
``` 
git remote -v
```

##### Change URL
```
git remote set-url origin https://github.com/yemyintsoe/repository-1.git
```


Ref: https://kodekloud.com/blog/change-remote-origin-in-git/
